## Hello and Welcome!

Welcome to `embed-dossier-mstr-react`, a package that allows you to embed MicroStrategy Dossier in your React application based on MicroStrategy Embedding SDK. This documentation provides a comprehensive guide to help you get started with the package, covering installation, authentication, configuration, and usage.

MicroStrategy's Embedding SDK Playground is the one-stop shop for developers to build impactful, interactive analytics experiences that integrate seamlessly with websites and applications.

Just use the intuitive drag-and-drop interfaces to define the perfect user experience, test it out in real time, and then let the platform automatically generate the code for you. You'll turn your application into platform for Intelligence Everywhere in no time.

But when implementing the Embedding SDK in your React application, you may face some challenges. That's where `embed-dossier-mstr-react` comes in. It provides a simple and easy-to-use package that allows you to embed MicroStrategy Dossier in your React application with minimal effort.

## Why Use `embed-dossier-mstr-react`?

`embed-dossier-mstr-react` is a powerful package that simplifies the process of embedding MicroStrategy Dossier in your React application. It provides a set of components and utilities that allow you to easily integrate MicroStrategy Dossier into your application, without having to worry about the underlying complexities of the Embedding SDK. Here are some of the key benefits of using `embed-dossier-mstr-react`:

- **Simplified Integration**: `embed-dossier-mstr-react` provides a set of components and utilities that make it easy to embed MicroStrategy Dossier in your React application.
- **Customization**: `embed-dossier-mstr-react` allows you to customize the appearance and behavior of the embedded Dossier to suit your needs.
- **Performance**: `embed-dossier-mstr-react` is optimized for performance, ensuring that the embedded Dossier loads quickly and runs smoothly.
- **Security**: `embed-dossier-mstr-react` handles authentication and authorization based on the MicroStrategy Embedding SDK, ensuring that your data is secure.
- **Ease of Use**: `embed-dossier-mstr-react` is designed to be easy to use, with clear documentation and examples to help you get started quickly.
